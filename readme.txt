# Hesap Makinesi

Bu basit hesap makinesi uygulaması, tkinter kütüphanesi kullanılarak Python'da oluşturulmuştur. 

## Özellikler

- Toplama
- Çıkarma
- Bölme
- Çarpma

## Gereksinimler

- Python 3.x
- tkinter kütüphanesi (genellikle Python ile birlikte gelir)

## Kullanım

1. Uygulamayı çalıştırmak için `hesap_makinesi.py` dosyasını çalıştırın.
2. İlk sayıyı "Sayı 1" giriş kutusuna girin.
3. İkinci sayıyı "Sayı 2" giriş kutusuna girin.
4. İşlem yapmak istediğiniz düğmeye tıklayın.
5. Sonuç etiketi altında işlem sonucunu görebilirsiniz.
6. "Temizle" düğmesine tıklayarak giriş kutularını temizleyebilirsiniz.

## Ek Bilgiler

- Eğer ikinci sayı sıfıra bölünmeye çalışılırsa, program "Bölen sıfır olamaz!" uyarısı verecektir.
- Giriş kutularına sadece sayılar girilmelidir. Başka bir veri türü girildiğinde program hata verecektir.

Bu basit uygulama, Python'u ve tkinter kütüphanesini kullanarak GUI (Grafiksel Kullanıcı Arayüzü) geliştirmenin temel bir örneğidir.
